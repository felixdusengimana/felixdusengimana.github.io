Jekyll is a simple, blog-aware, static site generator perfect for personal, project, or organization sites. You can think of it as a file-based CMS, without all the complexity. Jekyll takes your content, renders Markdown and Liquid templates, and spits out a complete, static website ready to be served by any web server.

## Why Jekyll?

- **Simple**: No databases, comment moderation, or pesky updates to install
- **Static**: Markdown, Liquid, HTML & CSS go in. Static sites come out ready for deployment
- **Blog-aware**: Permalinks, categories, pages, posts, and custom layouts are all first-class citizens

## Getting Started

To get started with Jekyll, you'll need:
- Ruby 2.7.0 or higher
- Bundler
- Jekyll

Once you have these installed, you can create a new Jekyll site and start building!

```bash
jekyll new my-site
cd my-site
bundle exec jekyll serve
```

Your site will be available at `http://localhost:4000`.

Happy Jekyll-ing!
